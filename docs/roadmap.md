# Roadmap

## Near term

- Release automation and GitHub release templates
- Optional analytics adapters
- Better diagnostics reporting modes for site owners

## Mid term

- Hosted per-site configuration service
- Signed remote config responses
- Rollout channels and version pinning strategies

## Longer term

- Commercial control plane integrations
- More advanced diagnostics export and developer tooling
- Additional presets and theming APIs without increasing core runtime weight
