## [0.1.0] - 2026-03-26

### Added

- Initial public release of the OurA11y embeddable accessibility enhancement library
- ESM and IIFE bundles for bundler and script-tag delivery
- Accessible launcher and control panel with persisted user preferences
- User-facing enhancements for readability, focus visibility, motion, contrast, cursor size, and reading guide
- Conservative remediation engine for safe low-risk DOM and ARIA improvements
- Optional diagnostics scanner for common detectable accessibility issues
- SPA-aware mutation observation and route change handling
- Example integrations, tests, and publication-ready documentation
